# sleep_and_lifestyle_health_analysis_report
Sleep & Lifestyle Health Analysis | SQL + Excel + Power BI

This project explores how sleep habits, stress levels, physical activity, and health indicators are connected.
Using a dataset of 374 records, I cleaned the data, built SQL insights, and created an interactive Power BI dashboard to understand health patterns.
